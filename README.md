# Naman Jain - Portfolio

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fnamanviber%2Fnj-portfolio)

This is the personal portfolio website of **Naman Jain**, a Member of Technical Staff and Software Developer specializing in App & Protocol-level development (QUIC), System Design, and AI Agent workflows.

The site is built with **Next.js 15+ (App Router)** and designed to be minimal, fast, and aesthetically pleasing with a dark-themed UI and smooth animations.

## âœ¨ Features

-   **Modern UI/UX**: Clean dark mode with vibrant "Electric Cyan" and "Neon Purple" accents.
-   **Interactive Elements**:
    -   **Rotating Tech Circle** in the Hero section.
    -   **Accordion-style** Experience section.
    -   **Grid Layout** for Projects with a "Show Less" expandable banner.
    -   **Hover Effects** on all cards and links.
-   **Fast & SEQ Optimized**: Built on Next.js App Router with server-side rendering and optimized metadata.
-   **Responsive**: Fully fluid layout adapting to mobile, tablet, and desktop screens.
-   **Smooth Animations**: Powered by `framer-motion` for page transitions and element reveals.

## ğŸš€ Getting Started

To run this project locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/namanviber/nj-portfolio.git
    cd nj-portfolio
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open locally**:
    Navigate to [http://localhost:3000](http://localhost:3000) in your browser.
