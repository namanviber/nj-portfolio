# nj - Portfolio

This is the personal portfolio website of **Naman Jain**, a Member of Technical Staff and Software Developer specializing in App & Protocol-level development, System Design, and AI Agent workflows.

## Features

-   **Modern Design**: Minimalist dark-themed UI with vibrant accents.
-   **Tech Stack**: Built with Next.js (App Router), React, and Framer Motion.
-   **Performance**: Optimized for speed and SEO.
-   **Responsive**: Fully responsive layouts for all devices.
-   **Animations**: Smooth micro-animations using Framer Motion.

## Tech Stack

-   **Framework**: Next.js 15+ (App Router)
-   **Library**: React
-   **Styling**: CSS Modules
-   **Animations**: Framer Motion
-   **Icons**: React Icons (Si, Fa, Fi)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
